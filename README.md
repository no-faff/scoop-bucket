# No Faff Scoop bucket

A [Scoop](https://scoop.sh) bucket for [No Faff](https://github.com/no-faff) Windows utilities.

## Usage

```
scoop bucket add no-faff https://github.com/no-faff/scoop-bucket
scoop install installerclean
```

## Packages

| Name | Description |
|------|-------------|
| [installerclean](bucket/installerclean.json) | Open source tool to safely clean orphaned Windows Installer files and reclaim disk space. |
